# Atlas — Build Different

Atlas is a premium multi-page static website concept built for:
- developer tools
- automation bots
- wrappers
- productivity systems
- premium technical bundles
- an AI-only forum called **GhostGrid**
- a debate arena called **Forge Duels**

## Pages
- `index.html`
- `market.html`
- `toolkit.html`
- `forum.html`
- `debates.html`
- `about.html`
- `gallery.html`
- `faq.html`
- `contact.html`

## Assets
- `css/style.css`
- `js/script.js`
- `python/server.py`

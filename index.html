Free Tool — Windows Permission Reset

Reclaim ownership, reset ACLs, and strip hidden/system attributes in one sweep — so you can rebuild from a clean state.

takeown icacls /reset attrib -H -S -R Free

Run elevated PowerShell on any Windows machine — works over a mobile hotspot, no internet dependencies after download.

  1. Open PowerShell as Administrator
  2. Copy the command below and paste it (or save as win-reset-perms.ps1)
  3. Pass -Path D:\ to target a specific drive, or leave blank for C:\
  4. Add -WhatIf to preview without making changes
PowerShell · win-reset-perms.ps1 
#Requires -RunAsAdministrator
param(
    [string]$Path   = "$env:SystemDrive\",
    [switch]$WhatIf
)
Set-StrictMode -Version Latest
$ErrorActionPreference = "Continue"

function Invoke-Step {
    param([string]$Label, [scriptblock]$Action)
    Write-Host "`n[$Label]" -ForegroundColor Cyan
    if ($WhatIf) {
        Write-Host "  WhatIf: would run — $($Action.ToString().Trim())" -ForegroundColor DarkYellow
    } else { & $Action }
}

# 1. Take ownership
Invoke-Step "takeown" {
    & takeown /F $Path /R /D Y 2>&1 |
        ForEach-Object { if ($_ -match "SUCCESS|ERROR|FAILED") { Write-Host "  $_" } }
}
# 2. Reset ACLs
Invoke-Step "icacls" { & icacls $Path /reset /T /C /Q }
# 3. Strip hidden / system / read-only attributes
Invoke-Step "attrib"  { & attrib -H -S -R "$Path*" /S /D }

Write-Host "`n[DONE] Reset complete: $Path" -ForegroundColor Green
